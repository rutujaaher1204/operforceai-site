
# OperforceAI Site (React + TypeScript + Tailwind)

## Quickstart
1. Install deps
   ```bash
   npm install
   ```
2. Run dev server
   ```bash
   npm run dev
   ```
3. Build
   ```bash
   npm run build && npm run preview
   ```

Built with Vite + React + TypeScript + Tailwind. The gradient background and copy mimic the provided screens.
